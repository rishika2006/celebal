# Superstore Sales Analysis 

##  Objective
Analyze Superstore dataset using SQL queries for business insights.

## Tools Used
- MySQL
- SQL Workbench
- GitHub

##  Key Analysis Performed
- Data exploration
- Filtering (WHERE conditions)
- Aggregations (SUM, AVG)
- Category-wise analysis
- Top customers & products
- Monthly sales & profit trends
- Data validation checks

## Key Insights
- Technology category has highest sales
- West region performs best in revenue
- Several products generate negative profit
- Sales show monthly variation trends

## Skills Demonstrated
- SQL querying
- Data aggregation
- Business analysis
- Data cleaning validation

##  Outcome
This project helps understand real-world retail sales patterns using SQL.
